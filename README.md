## Tooling philosophy

I like tools that are:

- CLI-first and scriptable,
- useful before they are polished,
- safe by default around irreversible state,
- agent-friendly without requiring an agent for deterministic work,
- extracted from real workflow pain rather than invented in a vacuum.

Forklift is the first public example of that pattern: a problem from maintaining a customized Hermes fork turned into a general tool other people can use.
